PhyDSL Download

PhyDSL is divided in two main components, a syntax directed editor in the form of an Eclipse plugin for gameplay design, and a standalone code generator package.

Prerequisites

- The PhyDSL plugin has as a prerequisite XText 2.9. You can download the XText plugin here.

- The PhyDSL generator requires Java 6 or above

- The PhyDSL Eclipse plugin and Generator can be used in Windows and Mac operating systems.

Download Eclipse Plugin

In order to install the PhyDSL plugin in Eclipse, open your Eclipse distribution:

Click Help > Install New Software... > Add.. > Give a name to the repository such as “PhyDSL” > In location, use the official PhyDSL plugin download site: http://hypatia.cs.ualberta.ca/~guana/phydsl/

Select all the plugins inside the PhyDSL category and Click Next. Eclipse will download the plugin, and install it > Please restart your Eclipse instance.
